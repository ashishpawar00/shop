# PROJECT: Laxmi Krashi Kendra (LKK)

## Vision
To provide India's farmers with a premium, AI-powered agricultural ecosystem that builds trust through data-driven advisory and seamless commerce.

## Core Values
- **Accessibility**: Multi-language support (Hindi/English) and intuitive, mobile-first design.
- **Trust**: Visible social proof, secure transactions, and verified expert advisory.
- **Innovation**: Leveraging AI for crop diagnostics and 3D visualization for product engagement.

## Tech Stack
- **Frontend**: Next.js 14, Tailwind CSS, Framer Motion, Three.js (React Three Fiber).
- **Backend**: Node.js, Express, MongoDB (Mongoose).
- **Storage**: Cloudinary (Media/Images).
- **Design Strategy**: "No-Line" Glassmorphism (Surface Deep -> Surface Highest).

## Ecosystem Integration
- **Framework**: GSD (Get Stuff Done) Lifecycle.
- **Intelligence**: 1,455 specialized agentic skills.
- **Design**: Stitch Unified Design System.

## Project Structure
- `/client`: Next.js frontend.
- `/server`: Node/Express backend.
- `/docs`: Project documentation and GSD control files.
